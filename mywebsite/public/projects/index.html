Projects

  • Zipf’s Law: A project to validate the rank-frequency distribution of the English language, given a large enough sample of text. This python program starts at the Wikipedia page for “Team sport” and follows the first link in each page, provided it has not been visited before. It then collates all the text, creating a word-frequency dictionary which can then be plotted. Repository.
